# Contributing to Vigilum Codex MVP

Thank you for your interest in contributing to our project.

## How to Contribute
1. Fork this repository.
2. Create a new branch: `git checkout -b feature/your-feature`.
3. Follow the commit style: Conventional Commits (e.g., `feat(ui): add new dialog`).
4. Ensure all Python code is formatted and passes syntax checks via `pyright`.
5. Open a Pull Request for review.

## Coding Guidelines
* Always provide docstrings for new functions.
* Keep context dependencies local and avoid hardcoded paths.
* Validate modifications before submitting.
