# Support Guidelines

Thank you for using the Vigilum Codex MVP modules. Please refer to the guidelines below for technical support.

## Getting Help
Before seeking support:
1. Verify that your system dependencies are met (e.g., `pyright` for LSP checks, `playwright` for Web Raider crawling, `zenity` for GUI prompts).
2. Execute the local diagnostic scripts to see if the error is due to missing packages or wrong path environments.
3. Review the module-specific `README.md` files.

## Support Channels
As a public MVP, active support is handled as follows:
* **Documentation Issues**: Open an issue requesting clarification on the schema or configuration setup.
* **Feature Adjustments**: If a script needs changes to support your environment, submit a pull request on a work branch matching the *Contributing* guidelines.
* **Contact**: Address inquiries through the official platform tags for `@lordmahonheim-bot`.
