# Security Policy

## Supported Versions
Security updates are actively applied to the following versions of the MVP codebase:

| Version | Supported |
| :--- | :--- |
| 1.x.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability
We take security vulnerabilities seriously. To report a security vulnerability or coordinate disclosure:
1. Do NOT open a public GitHub issue.
2. Contact the maintainers privately via the official channels designated by `@lordmahonheim-bot`.
3. Provide a detailed report, including steps to reproduce, impact assessment, and potential mitigations.

We will review your submission and respond within 48 hours to coordinate remediation and release.
